# Trae国际版

本文档用于介绍如何在 ChatBox 中安装Promptx MCP

## 安装命令

### 本地模式(推荐)

```shell
npx -f -y --registry https://registry.npmjs.org dpml-prompt@beta mcp-server
```

### Http模式

`http://localhost:3000/mcp`
