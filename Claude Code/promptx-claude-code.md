# Claude Code

本文档用于介绍如何在Claude Code中安装Promptx MCP

## 安装命令

### 本地模式(推荐)

```json
# 全局安装promptx
npm install -g dpml-prompt@beta

# 添加promptx mcp服务

```

### Http模式

```bash
claude mcp add --transport http promptx http://localhost:3000/mcp
```
